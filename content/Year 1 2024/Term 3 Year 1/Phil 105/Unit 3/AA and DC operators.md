# AA (affirm anteccedent)
- if a=>c
	- is A
		- is c

# DC (deny consequent)
- if a => c
	- is not c
		- is not a
- #### (if A then C), (a only if C)
	- Not C 
		- Not A
- #### tips
	- Dont use only if
		- it can create a fallacy
			- *the better way*=> (original thing) **only if** (the DC)
- ### Ex
- Cats  have free will only if they can vote
		- cats cant vote
		- cats dont have free will
